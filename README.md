# How to run...
Run he below commands :
# 1. clone the repo:

```
git clone https://github.com/krishnanunni6707/optics.git
```
# 2. create a vertual environment:
```
python -m venv venv
venv\Scripts\activate
```
# 3. install dependencies:
```
pip install -r requirements.txt
```
# 4. train the model
```
python train.py
```
# 5. predict the model
```
python predict.py
```
